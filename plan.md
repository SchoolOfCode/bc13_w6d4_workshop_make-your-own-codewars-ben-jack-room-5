# Nightclub Array

## Situation

### You're a nightclub owner and you're having trouble keeping under 18's out. Write a function that removes all of the people under the age of 18.

```js
// input
a = [
  { name: "James", age: 25 },
  { name: "John", age: 17 },
];
// output
b = [{ name: "James", age: 25 }];
```
